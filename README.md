1. First Clone Project.

```cmd
git clone https://github.com/usama-akram-gt/disease-detection.git
```

2. Install dependencies.

```cmd
npm install
```
3. Start the project for android

```cmd
npm start android
```
4. Start the project for iOS

```cmd
npm start ios
```

5. You're good to go!
